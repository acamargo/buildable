=Buildable

Precisa buildar o seu projeto e não sabe como?

Basta seguir o seguintes passos:

==Instalação

 Inclua a gem no Gemfile do seu projeto:

      group :development do
        gem 'buildable'
      end

 Instale a gem via bundle:

    $ bundle install

==Configuração

 Crie os arquivos de configuração

     $ bundle exec buildable init

 Edite o arquivo .buildable.yml para configurar a build para o seu projeto.

 Edite o arquivo Procfile para especificar como o seu projeto deve subir, lembrando
 que você pode criar configurações diferente para desenvolvimento e produção.

==Gerando Pacote

 Execute o seguinte comando

     $ bundle exec buildable build

 O .deb será criado no diretório _pkg_.

=Subindo o Projeto em QA/HOMOL/PROD

 Utilize o script de inicialização criados na instalação

    # service app_name-production start
